Flexbox

Flexbox Froggy
Flexbox - це цілий модуль, що містить у собі набір властивостей та їх значень як для батька flex-контейнера, так і для його дочірніх елементів flex-елементів.

Властивості FLEX – елементів. Застосовуються для flex-елемента, дочірнього об'єкта flex-контейнера

flex-grow - 0 заборонено. 1 дозволено. (за замовченням - 0) Визначає можливість елемента збільшуватись у розмірі, при необхідності заповнюючи весь flex-контейнер.

Button
flex-shrink - 0 заборонено. 1 дозволено. (за замовченням - 1) Визначає можливість елемента зменшуватись у розмірі, при необхідності заповнюючи весь flex-контейєр.
Button
flex-basis - Визначає базовий розмір елемента за умовчанням, до розподілу простору, що залишився. Значення можуть бути у відсотках пікселях тощо. auto - розмір по контенту
